# Code of Conduct

## Standard

This project follows the spirit of the Contributor Covenant: be respectful, assume good faith, and keep discussion focused on improving the project.

## Expected Behavior

- Use welcoming and inclusive language.
- Disagree with ideas, not people.
- Keep feedback specific, actionable, and evidence-based.
- Avoid harassment, insults, sexualized language, or personal attacks.
- Do not publish private information, secrets, private prompts, customer data, or private repository paths.

## Enforcement

Maintainers may edit, hide, or remove issues, comments, pull requests, or other contributions that violate this code of conduct. Repeated or severe violations may lead to a ban from the project.

For sensitive conduct reports, avoid posting private details publicly. Use the repository owner's private contact path if available, or open a redacted issue asking for a private channel.
